# Skenario 02 - Slow HTTP Terkontrol

Tujuan: menguji indikator Slow HTTP/Slowloris secara terbatas terhadap target VM lokal.

Parameter harus kecil dan terukur: koneksi, rate, dan durasi dibatasi. Hentikan jika target tidak responsif atau resource usage tidak wajar.

Artefak wajib: pcapng, Snort log, dan target metrics CSV. Label `Slowloris Detected` hanya valid jika evidence gate mengizinkan dan indikator pendukung tersedia.
