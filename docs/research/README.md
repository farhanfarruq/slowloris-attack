# Dokumentasi Riset VM Lokal Slow HTTP

Direktori ini berisi dokumentasi riset yang sudah disanitasi untuk laporan skripsi. Dokumen tidak menyimpan password, API key, token, atau credential SSH.

## Isi

- `project-inventory-2026-06-12.md` - kondisi awal project dan keputusan perapihan.
- `vm-local-runbook.md` - alur eksperimen VM lokal bertahap.
- `experiment-report-2026-06-12.md` - laporan hasil sesi saat ini dan status eksekusi.
- `scenarios/` - definisi skenario baseline, Slow HTTP terkontrol, HTTP burst, port scan terbatas, dan iPerf.

## Prinsip

Eksperimen hanya untuk VM lokal. Target publik, domain publik, scanning internet, bypass, persistence, dan privilege escalation tidak termasuk scope riset.
